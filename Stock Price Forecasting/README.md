Stock Price Forecasting: ARIMA, LSTM & GRU
Group project completed as part of a Master's-level course.
Objective: Compare classical and deep learning approaches to stock price forecasting across multiple tickers.
Description: This project applies statistical time series modeling and deep learning to forecast adjusted close prices for TSLA, GOOG, and NVDA. The pipeline covers data ingestion from Yahoo Finance and Alpha Vantage, stationarity testing, model training, and forecast evaluation. A MACD technical indicator was also computed and visualized for supplementary analysis. Models are compared using RMSE on TSLA close price data.
This repository is shared for academic reference and archival purposes. No claim of sole authorship is made.
Key Skills / Tools:

Python
Statistical time series modeling (ARIMA, ADF/KPSS, Ljung-Box diagnostics)
Deep learning for sequence forecasting (LSTM, GRU – PyTorch)
Financial data APIs (Yahoo Finance, Alpha Vantage)
Model evaluation (MAE, RMSE, MAPE, R², Directional Accuracy)
